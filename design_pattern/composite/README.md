# Composite pattern

内包しているものと、されているものが共通のクラスとして扱える。  
自分の親クラスと、クラスのメンバとして持っているクラスが同じ。

`composite -> component -> component -> component -> ...`

ッて感じで再帰的に扱えるようになる。
